Programming

In the programming unit, we have been introduced to Java. So far, I have been working on a portfolio of tasks which uses different skills that I have been picking up throughout the course. I have also learnt good design methodology using top-down design and how to write code that can be easily re-used in the future. At first to keep things simple, we using an IDE called Processing, which has purely graphical output. After about 8 weeks of using Processing, we started learning how to code console applications. For this we used the Eclipse IDE.


For my portfolio tasks, the techniques that I have implemented are: using variables for reusabilty of code, game modes to have differnt scenearios for simple games, using classes and also building a simple calendar using enumerated types.


I found this unit enjoyable as it required a lot of problem-solving. It was tricky, at times, especially when a new idea was introduced, but, after sticking with it, I eventually picked it up.
