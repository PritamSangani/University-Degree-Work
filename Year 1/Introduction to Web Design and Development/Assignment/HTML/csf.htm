Computer Systems Fundamentals

This unit has been split in two: one part is focused on Maths and the other is focussed on Logic. In CSF Maths, we have learnt three topics so far. These are: Sets, Matrices & Vectors and Functions. For Logic, we started off by learning about logic gates before moving on to Assembly Language


We have been assessed in this task with two different pieces oc coursework. For Logic we had to write a programme using Assembly Language and in Maths we had to answer questions on topics that we had learnt.


I found this unit enjoyable as I had done a lot of the Maths before, but, at first I found the Logic part quite hard.
